# File-Handling
A python project that handle your files in a given directory.
## How To Run
First add the 'src' to 'PYTHONPATH'. If you don't know how to do it, write this command in your terminal:
"""
export PYTHONPATH=${PWD}
"""
Then run:
"""
python src/main.py
"""
